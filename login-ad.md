# What's New: May 2026 Release Notes### New Features* **TrackAbout Web » Order Scheduler:** Schedulers can now access a new **Order Scheduler** page to plan and manage truck assignments across day, week, and month views.   * **Calendar Views**: The day view displays trucks as columns and hours as rows. The month view shows a full calendar grid for spotting busy or empty days.  * **Unassigned Orders Panel**: Displays orders at the top of the page that still need to be scheduled.  * **Drag and Drop Scheduling**: Drag orders from the **Unassigned Orders** panel to a truck row and time slot, drag existing blocks to reschedule, or resize a block to adjust its duration.  * **Order Details Popover**: Select an order block to add or remove assignments, view scheduled hours, unassign all resources, or open the full order detail page in a new tab.  * **Sidebar**: Includes a mini calendar for date navigation, an **Add Blocked Time** button for PTO, maintenance, or other unavailability, and an Add Holiday button for company-wide holidays.  * **Filters and Grouping**: Includes **Filters** and **Grouping** options to narrow the calendar to relevant trucks and orders.   * **Settings**: Lets schedulers configure work hours, weekend visibility, default order duration, and calendar column width. All settings are saved per user and restored on the next sign-in.* **Customer Portal » Order Search » Pending Orders Now Visible in the Customer Portal:** The **Order Search** page now displays all pending orders in one place, including orders submitted through TAM7 and orders imported from your ERP system. Each order shows the order number, department, planned delivery date, delivery status, order origin, and who entered it. For delivered orders, a link to the associated delivery record is available.### Enhancements* **TrackAbout Web » Integration Messages » Pagination Added to Integration Messages Grid:** The **Integration Messages** page now displays messages in pages instead of a single large list. Users can browse results in manageable sets, navigate between pages using the **Previous** and **Next** buttons or page number controls, and see the total number of messages at a glance. The grid also supports selecting a preferred page size. This improvement makes the page load faster and makes it easier to find and review specific messages.* **TrackAbout Web » Delivery Receipts**: Support for FTP Archiving of Additional Receipt Types: All delivery receipt types can now be archived to an FTP site every time a delivery is completed. Contact TrackAbout Support to configure this.* **TrackAbout Web » Setup » Barcode Range Rules**: Variable Length and Custom Prefix Now Supported: Barcode range rules now support variable-length barcodes and custom prefixes, giving administrators greater flexibility when configuring validation rules for different clients and barcode formats.### Fixes* **TrackAbout Web » Login » Expired Password Can Now Be Changed Without TrackAbout Support Assistance:** Users with an expired password can now successfully change their password directly from the **Change Password** page upon login. Previously, selecting **Save** on the **Change Password** page returned an error and instructed the user to contact their system administrator, creating a situation where the user could not log in or reset their own password. After entering the old password and a valid new password and selecting **Save**, the system now updates the password, automatically logs the user in, and redirects them to the home page.* **TrackAbout Web » Auto Renew Asset Agreements: Asset Agreements Now Renew Correctly When Balance is Fully Covered**: The **Auto Renew Asset Agreements** job now correctly renews asset agreements when the agreement quantity matches the customer's full balance. It automatically renews expiring asset agreements based on a configurable set of rules. This reduces the amount of manual labor each month before rental is run. Please contact TrackAbout Support for more information.
